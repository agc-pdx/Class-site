Restrictions
  • Use both HTML and CSS
  • Only use the HTML allowed in Levels 1 and 2, now adding
    •      <div>
    •      <hr>(horizontal rule)
    •      <marquee>(animating text side to side or vertically)
  • Only use the following CSS properties:
    •      padding
    •      margin
    •      border
    •      display
    •      float, clear
    •      height, width
    •      text-transform
    •      transform: rotate()
    • Goals
    Focus on positioning elements using CSS. Experiment with space and possibly try a very tall page or a very wide page which utilizes horizontal scroll.
