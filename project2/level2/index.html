Level Two
Restrictions
  • In addition to the elements in Level 1, you can use any of the following HTML elements:
    •      <span> (inline element)
    •      <h1> <h2> <h3> <<h4> <h5> <h6> (headings)
    •      <strong> (strong, i.e. bold, text)
    •      <em> (emphasized, i.e. italic, text)
    •      <ol> <ul> <li> (lists: ordered list, unordered list, list item)
    •      <a> (anchor links only, no linking to external sites)

    • Goals
    Focus on presenting the text in different hierarchies and scales using only HTML.
