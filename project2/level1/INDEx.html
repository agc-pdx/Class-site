Level One
Restrictions
  • In addition to the less-than sign <html>, <head>, <title>, and <body> tags, use these HTML elements (no CSS of any kind):
    •      <p> (paragraph)
    •      <br> (line break)
  • You can use &nbsp; which is equivalent to a visible space (this unicode stands for “non breaking space.”) This is normally used to create a space in a line that cannot be broken by word wrap

    • Goals
    Focus on breaking up the text in different ways. How does the poem originally look? What happens if you present it more spaced out? More condensed? In a specific shape or create a pattern out of it?
